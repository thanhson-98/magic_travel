Nuxt Installation

Please follow the instructions in the video to see how you can install Nuxtjs on your hosting:

  1. For local host -
    1. Open you command prompt
    2. npm install
    3. npm run dev (will start the dev server at http://loaclhost:3000)
  2. Deploying to Netlify - https://www.youtube.com/watch?v=TbQ4kWX29lQ


  1. For locally build -
    1. Open you command prompt
    2. npm run build (will build locally and generate public directory)


Other Scripts
  1. Open you command prompt
  2. yarn lint:fix (This will format your code and fix auto fixable eslint error)
